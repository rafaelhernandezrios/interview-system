# Email - Configuración para Gmail
# IMPORTANTE: Para usar Gmail, necesitas crear una "Contraseña de aplicación" (App Password)
# 
# Pasos para configurar Gmail:
# 1. Ve a tu cuenta de Google: https://myaccount.google.com/
# 2. Activa la verificación en dos pasos si no la tienes activada
# 3. Ve a: https://myaccount.google.com/apppasswords
# 4. Selecciona "Correo" y "Otro (nombre personalizado)" - escribe "Mirai Interview System"
# 5. Google generará una contraseña de 16 caracteres (sin espacios)
# 6. Usa esa contraseña en EMAIL_PASS (NO uses tu contraseña normal de Gmail)
#
# Ejemplo:
EMAIL_USER=tu_correo@gmail.com
EMAIL_PASS=abcd efgh ijkl mnop
# Nota: La contraseña de aplicación NO tiene espacios, usa: abcdefghijklmnop
#

# Google Calendar API - Para crear eventos automáticamente
# Pasos para configurar:
# 1. Ve a Google Cloud Console: https://console.cloud.google.com/
# 2. Crea un nuevo proyecto o selecciona uno existente
# 3. Habilita la API de Google Calendar
# 4. Crea credenciales OAuth 2.0 (Application type: Web application)
# 5. Descarga el JSON de credenciales
# 6. Obtén un refresh token usando OAuth 2.0 Playground: https://developers.google.com/oauthplayground/
#    - Selecciona "Calendar API v3" y scope: https://www.googleapis.com/auth/calendar
#    - Autoriza y obtén el refresh token
GOOGLE_CLIENT_ID=your_google_client_id
GOOGLE_CLIENT_SECRET=your_google_client_secret
GOOGLE_REFRESH_TOKEN=your_google_refresh_token
GOOGLE_CALENDAR_ID=primary
# Nota: GOOGLE_CALENDAR_ID puede ser 'primary' para el calendario principal o un ID específico de calendario
